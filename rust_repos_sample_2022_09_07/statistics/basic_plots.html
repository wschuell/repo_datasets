Diverse measures over time on the whole dataset¶

The following measures are plotted, for a time resolution defined in the cell below:

  • number of commits
  • number of new developers
  • number of active developers
  • number of new projects/repositories
  • number of active projects/repositories
  • number of downloads
  • number of stars
  • number of forks
  • number of sponsorships
  • number of issues / number of closed issues
  • number of pullrequests / number of merged pullrequests
  • cumulative number of developers
  • cumulative number of projects/repositories

Issues/Pull Requests survival rate¶

  • Probability of an Issue to still be open as a function of time
  • Probability of a Pull Request to still be unmerged as a function of time

At the end of the notebook one can also find two plots about the importance of Rust in repositories and users:

issues_df = pd.read_sql('''
WITH total_cte AS (SELECT COUNT(*) AS total_cnt FROM issues)
SELECT julianday(closed_at)-julianday(created_at) AS delay,
COUNT(*) OVER (ORDER BY julianday(closed_at)-julianday(created_at)) AS issue_count,
COUNT(*) OVER (ORDER BY julianday(closed_at)-julianday(created_at))*1./(SELECT total_cnt FROM total_cte) AS density,
1.-(COUNT(*) OVER (ORDER BY julianday(closed_at)-julianday(created_at))*1./(SELECT total_cnt FROM total_cte)) AS probability
FROM issues
WHERE closed_at IS NOT NULL
ORDER BY delay
;''',con=db.connection)

issues_df.plot(x='delay',y='probability')
plt.ylabel('Issues still open')
plt.xscale('log')
plt.xlabel('Delay (days)')
plt.ylim(bottom=0)
plt.show()

pr_df = pd.read_sql('''
WITH total_cte AS (SELECT COUNT(*) AS total_cnt FROM pullrequests)
SELECT julianday(merged_at)-julianday(created_at) AS delay,
COUNT(*) OVER (ORDER BY julianday(merged_at)-julianday(created_at)) AS pullrequest_count,
COUNT(*) OVER (ORDER BY julianday(merged_at)-julianday(created_at))*1./(SELECT total_cnt FROM total_cte) AS density,
1.-(COUNT(*) OVER (ORDER BY julianday(merged_at)-julianday(created_at))*1./(SELECT total_cnt FROM total_cte)) AS probability
FROM pullrequests
WHERE merged_at IS NOT NULL
ORDER BY delay
;''',con=db.connection)

pr_df.plot(x='delay',y='probability')
plt.ylabel('Pull Requests still unmerged')
plt.xscale('log')
plt.xlabel('Delay (days)')
plt.ylim(bottom=0.)
plt.show()

;''',con=db.connection)
